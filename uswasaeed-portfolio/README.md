# Uswah Saeed - Portfolio Website

This repository contains the code for Uswah Saeed's personal portfolio website, showcasing her work as a game developer. The website is designed to be a comprehensive landing page detailing her projects, education, experience, awards, volunteering, skills, and art & design work.

## Technologies Used

*   **HTML5:** For the structure and content of the web pages.
*   **CSS3:** Custom styling is provided via `assets/css/style.css`.
*   **Tailwind CSS:** A utility-first CSS framework is used for rapid UI development and responsive design. The Tailwind CSS JIT compiler is included via CDN.
*   **JavaScript:** For interactive elements, dynamic content, and features like the mobile navigation toggle and the recommendations carousel.

## Features

*   **Responsive Design:** The website is designed to be fully responsive and accessible on various devices, from desktops to mobile phones.
*   **Navigation:** A fixed sidebar navigation for larger screens and a toggleable mobile navigation for smaller screens.
*   **Dynamic Content:** Interactive elements such as a carousel for recommendations.
*   **Portfolio Showcase:** Detailed sections for professional and academic projects, game jams, art & design.
*   **Professional Overview:** Sections for education, work experience, awards, and volunteering.
*   **Skills & Tools:** A clear representation of technical proficiencies and tools used.

## How to View

To view the website, simply open the `index.html` file in your web browser. As it's a static site, no special server setup is required.

```bash
# Clone the repository (if not already cloned)
git clone <repository-url>
cd uswasaeed-portfolio

# Open index.html in your browser
# (e.g., on macOS)
open index.html
# (e.g., on Windows)
start index.html
```
